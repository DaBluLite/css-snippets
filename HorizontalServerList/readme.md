# DaBluLite's Horizontal Server List
A different impementation of Gibbu's original HSL snippet

## Importing (Bottom):
### Normal:
```css
@import url("https://dablulite.github.io/css-snippets/HorizontalServerList/import.css");
```
### Unified With User Area:
```css

```
